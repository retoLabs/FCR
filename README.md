MySQL:
Verificar estado de mysql:  
$ service mysql status

Arrancar:  
$sudo systemctl start mysql

Conectarse:  
$ mysql -u root -p  
Enter password: Pauet:21

Ver bases de datos:  
mysql> show databases;

Usar una BD:  
mysql> use pauet

Ver tablas de la BD en uso:  
mysql> show tables;

Ver estructura de una tabla:  
mysql> describe *tabla*;

Salir de mysql:  
mysql> quit
